# Línea 3.5
Jorge y Paloma son estudiantes de Comunicación Visual interesados en explorar cómo el diseño puede dialogar con problemáticas urbanas y sociales. A lo largo de este proyecto combinaron investigación, trabajo de campo y construcción visual para abordar el caso de la Línea 7 del Metrobús desde una perspectiva crítica sobre desigualdad y movilidad en la Ciudad de México
